Axis Data Limits:

This page allows you to set the physical limits of the axis drawn in the display area. The default configuration autoscales the display area to the full limits of all spectra that it is displaying.

If you want to apply limits of your own then, untick the autoscale checkboxes and enter your values in the lower and upper entry areas. Remember to press Apply to apply the limits.

Other useful options are the checkboxes Fit X range: and Fit Y range: that also make the ranges that you supply fit the display area exactly (these are like the fitwidth and fitheight options in the plot window) and Y auto cut: which applies limits to the Y axis using percentile cuts.

The Update to current button sets the X and Y limits to those of the full plot (these are the values shown initially). The Update from view button sets the X and Y limits to those of the visible part of the display area. This is useful to just show coordinate axes for what you can see.
