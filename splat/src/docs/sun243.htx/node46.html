SPLAT-VO is written in Java by SUN Microsystems Inc. and contains software from:

  • The JDOM Project (http://www.jdom.org/).
  • The Apache Software Foundation (http://www.apache.org/).
  • The DIVA project (http://embedded.eecs.berkeley.edu/diva/).
  • The COLT distribution (http://hoschek.home.cern.ch/hoschek/colt/).

  • TAMFITS - FITS data access library (http://heasarc.gsfc.nasa.gov/docs/heasarc/fits/java).
  • EPSGRAPHICS - Java EPS Graphics2D package (GPL version) (http://www.jibble.org/epsgraphics).
  • JETTY - Java Web server (http://jetty.mortbay.org/).

The BeanShell (remote control and scripting) is written by Pat Niemeyer (http://www.beanshell.org).

The flip toolbox and Hermitian interpolating polynomials are based on the similiar functionality in the SPEFO program, and were suggested by Petr Skoda (skoda@sunstel.asu.cas.cz, these features are particularly useful for hot star research, contact Petr for further details).

This work has mainly been supported by the now closed Starlink Project, funded by the Particle Physics and Astronomy Research Council and more recently by the Joint Astronomy Centre Hawaii. Finally, many programmers working for Starlink have made contributions to the components used in SPLAT-VO, and many users have reported problems, as well as requests for useful features - thanks to them all.
