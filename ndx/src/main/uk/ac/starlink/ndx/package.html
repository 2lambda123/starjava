Classes for using and implementing Ndx objects. In order to use Ndxs, only the {@link Ndx} and {@link NdxAccess} classes are required.

Exceptions

Use of exceptions is kept to a reasonable minimum for the Ndx class. In general the policy is that if something serious fails, such as an Ndx not being able to supply its Image array, a checked NdxException will be thrown. However if something minor fails, such as an Ndx not being able to supply its title, a sensible value (such as null) is returned and a message is logged through the uk.ac.starlink.ndx logger.
