Table Metadata Window

This window displays any metadata which describes the table as a whole, for instance its name, dimensions and any other parameters known. The metadata is displayed in a JTable widget, with one row per item of metadata. The important columns are Name, Value and Description, though other columns display other bits of information about each category - the Display menu can be used to reveal or hide these columns as desired.

Editing Metadata

Values and descriptions for some of the items can be edited. Double-click on them, and type the new value to change them. Some however, like number of columns, are read-only and cannot be edited in this way.
