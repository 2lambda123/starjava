Release Notes

This is TOPCAT, Tool for OPerations on Catalogues And Tables. It is a general purpose viewer and editor for astronomical tabular data developed by Starlink.

Author
Mark Taylor (Starlink, Bristol University)
Email
m.b.taylor@bristol.ac.uk
WWW

Version 0.2a

Released for the Starlink Summer(?) 2003 CD.

This is alpha software. It has enough functionality to be useful but has not been rigorously tested and needs additional functions, some improvements to the user interface and so on.
