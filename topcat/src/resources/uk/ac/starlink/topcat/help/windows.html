Windows

When running TOPCAT can display a number of windows on the screen, displaying different views of the table data and metadata. The principle one is the main browser window which is shown as soon as a table is loaded, but others can be popped up by using the items in the Windows menu or the corresponding buttons in the toolbar.

All the main windows have some features in common:

Close
The Close button on the toolbar or the "Close" item in the File menu will close the window. When the last window is closed the application should exit.
Exit
The "Exit" item in the File menu will exit TOPCAT straight away regardless of what windows are open.
Help
The Help button on the toolbar or the "Help browser" item in the Help menu will bring up the help browser (this window). If context-sensitive help is available from this point (e.g. help about the window in which the button was pressed) then the browser will come up displaying the relevant page.
About
The "About" item in the Help menu will pop up a short message about the TOPCAT application.
The main windows available are these:
Table browser window
Column metadata window
Statistics window
Plot window
Table metadata window
Help window
