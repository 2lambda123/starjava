Diva Release 0.2

Welcome

Diva is a software infrastructure for visualizing and interacting with dynamic information spaces. This documentation tree contains version-specific infromation about installation, demos, trouble-shooting, packages, etc. For general documentation, papers, etc. please see the Diva homepage.

This is release 0.2 of Diva. The release contains three main packages. canvas is a flexible structured graphics layer built on the Java2D API. graph package provides a Swing-style (MVC) graph viewing architecture. sketch is a framework for sketch (free-hand) input and gesture recognition. See the package page for details.
